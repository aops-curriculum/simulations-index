# AoPS Physics Simulations — Index

A searchable, filterable index of every interactive physics simulation across all AoPS
physics courses. **Live page:** https://aops-curriculum.github.io/simulations-index/

Each card links to the live GitHub Pages URL of the sim (in its own course repo) and has
a ⧉ button to copy that URL.

## This repo holds only the generated artifacts

- `index.html` — the search/filter/sort page (self-contained)
- `simulations.json` — aggregated sim data (titles, descriptions, dates, screenshots)
- `screenshots/` — preview thumbnails for non-539 sims
- `.nojekyll` — serve files as-is on GitHub Pages

## The generator lives in the parent container, not here

`github-aops/` is a container folder (not a repo). The tooling that builds this repo is at
`github-aops/scripts/` and must run there, because it scans the sibling course folders:

```
cd github-aops
npm run generate-index     # rebuild index.html + simulations.json (AI descriptions + screenshots)
npm run deploy             # rebuild, then commit & push this repo to GitHub Pages
```

### AI descriptions use the Claude subscription, not the API

New sims get a 1–2 sentence description generated by the **Claude Code CLI** (`claude -p`),
which bills against the Claude **subscription**. No Anthropic API key is used anywhere.

### Auto-update

`github-aops/scripts/watcher.js` (Windows Task Scheduler: "AoPS Sim Index Watcher") watches
all `###_*` course folders and rebuilds the **local** index automatically as sims are added,
edited, or deleted. Publishing to GitHub Pages is done with `npm run deploy`.
